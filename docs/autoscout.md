## Blockscout Autoscout (Optional Explorer)

Autoscout provides a self-hosted explorer with quick setup.

### Steps

1) Create an Autoscout account and new explorer.
2) Provide chain parameters:
   - RPC URL: your Sepolia endpoint
   - Chain ID: 11155111 (Sepolia)
   - Currency: ETH
3) Deploy; copy the explorer URL.

### MCP Analytics (Optional)

Integrate the MCP server for AI-powered analytics and insights:
- Whale trade detection
- MEV extraction patterns
- Bid/settlement timelines

### Monitoring

- Subscribe to hook events (`HookSwap`, `MEVDetected`) and channel events for dashboards.
